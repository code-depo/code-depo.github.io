Bubble sort

  • Starting from the beginning compare every adjacent pair of elements if they are not in the right order swap them.
  • At the end of one iteration the largest element is at the last position.
  • procedure is repeated no. of elements times.
  • In each iteration an element is added to the sorted part.
  • Illustration
    Intermediate bubbling(swaping) is abbreviated to just the final result.

  • This is a Θ(n2) algorithm.
